# 📊 Student Exam Score Analysis Dashboard

## 📌 Project Overview

This project presents an interactive **Student Exam Score Analysis Dashboard** built using **Power BI**. The dashboard analyzes student performance across various subjects and demographic factors to uncover trends, identify performance gaps, and support data-driven educational decisions.

The goal is to transform raw student assessment data into meaningful insights that help educators understand the key factors influencing academic performance.

---

## 🎯 Business Objectives

* Analyze overall student academic performance.
* Compare scores across different subjects.
* Identify factors that influence exam performance.
* Evaluate the impact of parental education, test preparation, and demographic characteristics.
* Enable interactive exploration through dynamic dashboards.

---

## 🛠️ Tools & Technologies

* **Power BI**
* **Power Query**
* **DAX (Data Analysis Expressions)**
* **Microsoft Excel / CSV**
* **Data Modeling**

---

## 📂 Dataset

The dataset contains student-related information, including:

* Gender
* Race/Ethnicity
* Parental Level of Education
* Lunch Type
* Test Preparation Course
* Math Score
* Reading Score
* Writing Score

---

## 📈 Key Performance Indicators (KPIs)

* 👨‍🎓 Total Students
* 📚 Average Math Score
* 📖 Average Reading Score
* ✍️ Average Writing Score
* 🏆 Overall Average Score
* ✅ Students Passed
* 📈 Pass Percentage
* 🎯 Test Preparation Completion Rate


---

## 📌 Business Insights

* Compared student performance across mathematics, reading, and writing.
* Evaluated the impact of completing a test preparation course on exam scores.
* Analyzed score variations by gender and parental education level.
* Identified factors associated with higher academic performance.
* Enabled educators to explore trends through interactive visualizations.


---

## 🚀 How to Use

1. Download the `.pbix` file.
2. Open it in **Power BI Desktop**.
3. Refresh the dataset if required.
4. Explore the dashboard using filters and slicers.
